⚡ WarOfPoints

WarOfPoints — это стратегическая игра на основе захвата точек на карте. Каждый игрок управляет армией, которая перемещается между точками, захватывая их и защищая свои территории.

Особенности:

  • Динамическая карта с изменяемыми границами
  • ИИ-противники с разными уровнями сложности
  • Режимы: кампания, быстрая игра, онлайн-мультиплеер (в планах)

Релиз запланирован на 2026 год. Следите за обновлениями!

🧠 MarkovTransform

MarkovTransform — нейросетевой проект, использующий марковские цепи для генерации музыки, текста и изображений.

Технологии:

  • Рекуррентные нейронные сети (LSTM)
  • Обучение с подкреплением для управления генерацией
  • Интерфейс для интерактивного творчества

Прототип уже работает. Ожидайте бета-версию в ближайшие месяцы.
